
Aspose.OCR for Java
==========================================================================

This distribution contains the following directories:

- lib      : the jars, compiled and ready to use
- docs     : the documentation in html format
- samples : some demos


For start please check docs/index.html



Required Software and Libraries
===============================

Note that you will have to install
- Java runtime 1.6 or above
- Log4j v1.2
  http://logging.apache.org/log4j/



Samples
========

- Check README.txt files from the respective demo folders for instructions on how to execute the demo.


Documentation
======================

- For the Aspose.OCR for Java API Reference in the JavaDoc format see docs\index.html.

- http://www.aspose.com/documentation/java-components/aspose.ocr-for-java/index.html is the home page for Aspose.OCR Product Documentation
that includes Getting Started, Programmer's Guide and other articles.
